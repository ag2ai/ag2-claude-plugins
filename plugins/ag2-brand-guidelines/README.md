# AG2 Brand Guidelines Plugin

Applies AG2's official design system and brand identity to all generated visual output — HTML, React components, landing pages, dashboards, presentations, and any UI code.

No Figma account or design tool access required. All design tokens are baked into the plugin.

## Install

### Claude Desktop (Cowork)

1. Go to **Directory → Plugins → Add marketplace**
2. Enter `ag2ai/ag2-claude-plugins` → click **Sync**
3. Find **AG2 Brand Guidelines** in the plugin list and install it
4. Start a Cowork session — the plugin is active

### Claude Code (CLI)

```bash
claude plugin marketplace add ag2ai/ag2-claude-plugins
claude plugin install ag2-brand-guidelines
```

## Use It

Just ask Claude to build anything AG2-related. The plugin activates automatically.

```
Build me an AG2 landing page
```
```
Create a dark AG2 dashboard with agent stats
```
```
Make a React component for an AG2 pricing page
```
